# EiT-Client

## Robotserver API
### Get Data
```HTTP
get localhost:<port>/data/<every-ms>
```
> `every-ms` specifies the interval (in millisec) of received data packages

## Dependencies
### Gorilla/mux
#####install: 
go get -u github.com/gorilla/mux
#####More docs:
